# Frontend (React + Vite + Tailwind)

## Setup

- `cd frontend`
- `npm install`
- Copy `env.example.txt` to `.env`
- Fill Supabase + API values
- `npm run dev`

Frontend runs at `http://localhost:5173`.

## Pages Included

- Home
- Login
- Signup
- Dashboard
- Score Entry
- Charity Selection
- Draw Results
- Admin Panel
